# 第二阶段：Agent 入门阶段（3-4 周）

## 核心目标
吃透 Agent 核心定义与四要素，掌握单智能体的完整开发流程，能基于 JS 生态实现端到端的可用 Agent，完成从 “LLM 调用” 到 “Agent 开发” 的核心跨越。

## 必修内容
1. Agent 核心认知扫盲
    - Agent 的核心定义、与普通 LLM 对话的本质区别
    - Agent 四大核心模块：规划 Planning、记忆 Memory、工具使用 Tool Use、行动 Action
    - 经典 Agent 范式：AutoGPT、BabyAGI、ReAct 核心原理与设计思路
    - JS 生态主流 Agent 框架：LangChain.js 核心入门（优先学习，贴合 TS 栈）、LlamaIndex.TS 基础认知
2. Agent 核心模块逐个突破（每个模块必须手写 Demo）
    - 工具调用 Tool Use（Agent 核心能力）
    - Function Calling 完整规范：工具定义、参数校验、调用执行、结果回传全流程
    - 自定义工具开发：天气查询、网页抓取、代码执行、数据库查询等基础工具开发
    - 多轮工具调用、工具错误重试、超时控制的基础实现
    - 记忆系统 Memory
    - 记忆分层：短期对话记忆、长期记忆的核心区别与使用场景
    - 对话上下文管理：历史消息裁剪、摘要生成、上下文窗口优化
    - 向量数据库实战：Chroma/Pinecone（二选一，优先 Chroma 本地部署）的 JS SDK 对接，实现长期记忆的存储与检索
    - 规划能力 Planning
    - ReAct 框架（推理 + 行动）完整实现：手写 ReAct Prompt、实现 “思考 - 行动 - 观察” 的循环执行逻辑
    - 基础任务拆解：长任务的分步规划、子任务执行与结果汇总
    - Web Agent 入门（前端开发者专属优势）
    - Puppeteer/Playwright JS SDK 入门，实现浏览器自动化操作
    - 开发基础网页操作 Agent：自动网页总结、表单填写、信息抓取
3. Agent 前端交互实现
    - Agent 思考过程、工具调用状态的实时前端可视化
    - 多轮对话的分支管理、历史回溯、编辑重发功能实现
    - 基于 Next.js 的 Agent 全栈项目架构设计
    
## 选修内容
- 低代码 Agent 平台入门：Dify、Coze 平台使用，理解 Agent 可视化编排逻辑
- 浏览器插件开发基础，实现简单的侧边栏 Agent 插件
- 多模态 Agent 入门：对接图片 / 语音模型，实现带文件上传的对话 Agent

## 实战项目 & 验收标准
- 实战：基于 Next.js + LangChain.js + 主流 LLM API，开发一个全栈个人助理 Agent
- 核心能力：多轮对话记忆、3 个以上自定义工具调用、ReAct 推理框架、流式对话界面、工具调用状态实时展示
- 验收：
  1. 能独立完成单智能体的完整开发，端到端跑通无阻塞
  2. 能稳定实现工具调用，处理工具执行异常，LLM 能根据工具返回结果完成推理
  3. 能讲清楚 Agent 四大模块的作用、协同逻辑，以及 ReAct 框架的执行流程
  4. 前端交互流畅，流式输出、状态同步无 bug
