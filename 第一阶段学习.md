# 第一阶段：前置筑基阶段（2-3 周）

## 核心目标
补齐 AI 核心基础，巩固前端全栈适配能力，搭建 Agent 开发的基础技术体系，避免后续学习出现 “空中楼阁”。
必修内容

## 一、前端全栈能力巩固（贴合 Agent 开发，快速过）

1. Node.js 进阶核心
    - 异步编程深化：Promise/async-await、事件循环、流处理、并发控制
    - 核心模块：fs/child_process/http/https，工具调用、系统集成必备能力
    - 服务端框架：Express/Nest.js（二选一，优先 Nest.js，企业级开发更友好）
    - 全栈框架：Next.js 深度入门（核心重点，前端开发 Agent 的首选框架，API 路由、服务端组件、流式渲染完美适配 AI 场景）
2. 前端交互核心适配
    - 流式通信：SSE、WebSocket、Fetch 流式响应（Agent 对话打字机、实时状态同步核心）
    - React Hooks 进阶（状态管理、异步请求封装、流式数据处理）
    - 工程化：Git、Docker、模块化 / 组件化设计、接口鉴权与限流
3. 必备工具库
    - Vercel AI SDK（前端开发 AI 应用的神器，极简对接全主流 LLM、流式输出、React Hooks 一站式支持）
    - Zod（TS 类型校验，工具调用参数校验核心）

## 二、AI 核心基础（前端开发者核心补全项，必须吃透）

1. LLM 大语言模型核心原理（无需深入数学，懂应用层原理即可）
    - Transformer 核心架构、Tokenization、上下文窗口、注意力机制核心逻辑
    - LLM 生成原理、采样参数（温度系数 Temperature、Top-P/Top-K）的调优逻辑
    - 主流 LLM 能力边界：OpenAI GPT 系列、Anthropic Claude、国内通义千问 / 文心一言 / DeepSeek 等模型的适配差异
2. Prompt Engineering 核心（Agent 的灵魂，贯穿全学习周期）
    - 基础能力：角色设定、零样本 / 少样本提示、指令优化、格式约束
    - 进阶推理：思维链 CoT、自我一致性、思维树 ToT、指令拆分
    - 核心规范：Function Calling 工具调用提示词、ReAct 框架提示词编写
3. 向量与检索核心基础
    - 嵌入 Embedding 核心概念、向量空间、余弦相似度计算逻辑
    - 向量数据库核心作用、基础 CRUD、检索原理
4. LLM API 实战
    - 主流 LLM 的 JS/TS SDK 对接（openai-node、通义千问 JS SDK 等）
    - 核心能力调用：对话补全、流式输出、Function Calling、多模态理解、嵌入生成
    - 接口错误处理、超时控制、Token 用量统计

## 选修内容
- Python 基础语法（仅做了解，无需深入，优先吃透 JS 生态）
- HTTP/HTTPS、RESTful API、GraphQL 进阶规范
- 基础的 Linux 命令、云服务器操作

## 实战项目 & 验收标准
- 实战：基于 Next.js + Vercel AI SDK，开发一个带流式对话界面的 LLM 聊天机器人，实现多轮对话、参数可调、多模型切换
- 验收：
  1. 能独立完成 LLM API 对接，流式输出无卡顿，对话上下文连贯
  2. 能通过 Prompt 优化，让 LLM 稳定输出指定格式的内容，理解 Function Calling 的核心逻辑
  3. 能讲清楚 LLM、Embedding、Token 的核心概念，无原理性认知错误
